# VaultCore Banking System

A full-stack banking transaction management system built using Flask, SQLite, HTML, CSS, and JavaScript.

## Features

- JWT Authentication
- Password Hashing using bcrypt
- Customer Management
- Account Management
- Deposit Funds
- Withdraw Funds
- Transfer Funds
- Transaction History
- Dashboard Analytics
- Responsive Banking UI

## Tech Stack

- Flask
- SQLite
- HTML
- CSS
- JavaScript
- JWT
- bcrypt

## Future Improvements

- Email OTP Verification
- Forgot Password
- PDF Bank Statements
- Admin Panel
- Charts & Analytics

pip install -r requirements.txt
